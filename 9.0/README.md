# .NET MAUI 9 Samples (End of Support)

> [!IMPORTANT]
> **.NET 9 has reached end of support.** These samples are no longer maintained.
>
> Please use the [10.0/](../10.0/) folder for the current stable .NET MAUI samples.

## Upgrading to .NET 10

To upgrade your .NET 9 app to .NET 10, see the official migration guide:

- [Upgrade from .NET 9 to .NET 10 - Microsoft Learn](https://learn.microsoft.com/dotnet/maui/migration/upgrade-to-net10)

## Previous samples

The .NET 9 samples were removed from this repository when .NET 9 reached end of support. If you need to reference the old code, check the git history or the [`net9-archive`](https://github.com/dotnet/maui-samples/tree/net9-archive) tag.
